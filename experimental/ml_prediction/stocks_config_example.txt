# Stock Data Configuration
# Format: STOCK_ID=GOOGLE_SHEETS_URL
#
# To use Google Sheets:
# 1. Open your Google Sheet
# 2. Get the shareable link or use File > Download > CSV to get export URL
# 3. For sheets with multiple tabs, add &gid=XXX to specify the tab
#    (gid=0 is first tab, gid=1 is second tab, etc.)
#
# Example URL format:
# https://docs.google.com/spreadsheets/d/SPREADSHEET_ID/export?format=csv&gid=TAB_NUMBER
